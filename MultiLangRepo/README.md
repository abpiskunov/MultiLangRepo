# MultiLang Sample Repository

A sample repository demonstrating a product built with multiple languages and frameworks.

## Structure

| Folder | Language | Projects |
|--------|----------|----------|
| `dotnet/` | .NET Framework 4.7.2 | eShopMvc (ASP.NET MVC), Utilities, DataAccess |
| `cpp/` | C++17 | string-utils, math-engine, file-processor |
| `typescript/` | TypeScript 5.x | task-manager, csv-parser, http-client |
| `python/` | Python 3.10+ | log-analyzer, config-manager, data-pipeline |
| `java/` | Java 11 | inventory-service, json-toolkit, sorting-demo |

## Purpose

This repository simulates a real-world product where different subsystems are implemented in the language best suited for their domain.
